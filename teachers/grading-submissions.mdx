Leopard provides a comprehensive grading workflow for reviewing student submissions, assigning grades, and providing feedback.

## Submission Schema

Each submission contains:

```typescript
// From: packages/backend/convex/schema.ts:37-51
submissions: {
  assignmentId: Id<"assignments">,
  flags: Id<"flags">[],
  grade?: number,
  studentId: string,                    // Better Auth user ID
  submissionFeedback?: string,
  workspaceId?: Id<"workspaces">,
  submittedAt: number,                  // Unix timestamp
  gradedBy?: string,                    // Better Auth user ID
  gradedAt?: number,                    // Unix timestamp
  gradesReleased: boolean
}

// Indexed by: studentId_assignmentId, assignmentId_studentId
```

## Grading API

### Assigning a Grade

Set a numeric grade for a submission:

```typescript
// From: packages/backend/convex/web/teacherAssignments.ts:215-238
await gradeSubmission({
  submissionId: "submission_id",
  grade: 95.5
});

// Updates the submission with:
{
  grade: 95.5,
  gradedBy: user._id,      // ID of teacher/TA who graded
  gradedAt: Date.now()     // Timestamp of grading
}
```

<Note>
  Grades are stored as `float64` values, allowing for decimal grades like 95.5 or 87.25.
</Note>

### Providing Feedback

Add written feedback for students:

```typescript
// From: packages/backend/convex/web/teacherAssignments.ts:241-262
await provideSubmissionFeedback({
  submissionId: "submission_id",
  feedback: "Great work on the algorithm implementation. Consider optimizing the space complexity in your next submission."
});

// Updates:
{
  submissionFeedback: "Great work on..."
}
```

Feedback is stored as plain text but can include:
- Specific code references
- Improvement suggestions
- Positive reinforcement
- Links to resources

## Grade Release Process

Control when students can view their grades:

```typescript
// From: packages/backend/convex/schema.ts:47
gradesReleased: boolean

// When true: students can view their grade and feedback
// When false: grade is hidden from students
```

<Warning>
  The grade release functionality is tracked in the schema but not yet fully implemented in the UI. Currently, grades are visible to students immediately upon being saved.
</Warning>

### Planned Grade Release Features

1. **Bulk Release**: Release all grades for an assignment at once
2. **Individual Release**: Release grades one student at a time
3. **Scheduled Release**: Set a future date/time for automatic release
4. **Conditional Release**: Release grades when all submissions are graded

## Submission Card Component

Complete example of the grading UI:

```tsx
// From: apps/nextjs/src/app/teacher/classroom/[id]/assignment/[assignmentId]/page.tsx:42-145
function SubmissionCard({
  submission,
  onSave,
}: {
  submission: Doc<"submissions">;
  onSave: (
    submissionId: Id<"submissions">,
    grade: number | null,
    feedback: string,
  ) => Promise<void>;
}) {
  const [isSaving, setIsSaving] = useState(false);
  const [grade, setGrade] = useState(submission.grade?.toString() ?? "");
  const [feedback, setFeedback] = useState(submission.submissionFeedback ?? "");
  
  const handleSave = async () => {
    const nextGrade = grade.trim() === ""
      ? null
      : Number.isNaN(Number(grade))
        ? Number.NaN
        : Number(grade);
    
    if (Number.isNaN(nextGrade)) {
      toast.error("Grade must be a valid number");
      return;
    }
    
    setIsSaving(true);
    try {
      await onSave(submission._id, nextGrade, feedback);
      toast.success("Submission updated");
    } catch (error) {
      toast.error(
        error instanceof Error ? error.message : "Failed to update submission"
      );
    } finally {
      setIsSaving(false);
    }
  };
  
  return (
    <Card>
      <CardHeader>
        <CardTitle>{submission.studentId}</CardTitle>
        <CardDescription>
          Submitted {new Date(submission.submittedAt).toLocaleString()}
        </CardDescription>
        {submission.grade !== undefined && (
          <div>{submission.grade}</div>
        )}
      </CardHeader>
      <CardContent>
        <div>
          <Label htmlFor={`grade-${submission._id}`}>Grade</Label>
          <Input
            id={`grade-${submission._id}`}
            value={grade}
            onChange={(event) => setGrade(event.target.value)}
            placeholder="e.g. 95"
            type="number"
            step="0.01"
          />
        </div>
        
        <div>
          <Label htmlFor={`feedback-${submission._id}`}>Feedback</Label>
          <textarea
            id={`feedback-${submission._id}`}
            rows={4}
            value={feedback}
            onChange={(event) => setFeedback(event.target.value)}
            placeholder="Feedback for the student"
          />
        </div>
        
        <Button onClick={handleSave} disabled={isSaving}>
          {isSaving ? "Saving..." : "Save Grade & Feedback"}
        </Button>
      </CardContent>
    </Card>
  );
}
```
